# Verdant Collapse — 테스트 체크리스트 & 밸런스 조정 포인트

## 1. 기능 테스트

### 코어 루프
- [ ] 플레이어 WASD/Gamepad 이동 정상 작동
- [ ] 이동 입력 없어도 자동 공격 발동
- [ ] 적 처치 시 XP 젬 정확한 위치에 스폰
- [ ] 젬 콜라이더 진입 시 흡수 애니메이션 후 XP 증가
- [ ] XP 바 가득 참 → 레벨업 UI 표시 & 게임 일시정지
- [ ] 3장 카드 선택 후 Playing 상태 복귀
- [ ] 20분 생존 시 Victory → Result 씬 전환
- [ ] HP 0 시 Dead → Result 씬 전환

### 무기
- [ ] Thorn Whip: 레벨 오를수록 회전 가시 수 증가 확인
- [ ] Spore Cloud: 범위 안 적에게 틱 데미지 적용
- [ ] Root Spike: 관통 수만큼 적 통과
- [ ] Petal Storm: 정확히 8방향 산탄
- [ ] Vine Lash: 가장 가까운 적 순서대로 히트
- [ ] Blight Bolt: 유도 작동, 화면 가장자리 적 추적

### 진화
- [ ] Thorn Whip MAX + Thick Bark MAX → 보물상자 스폰
- [ ] 플레이어 상자 접촉 → Briar Maelstrom 획득 & 원본 무기 교체
- [ ] 진화 후 기존 무기 슬롯 아이콘 변경 확인

### 적 스폰
- [ ] 0~3분: 일반 적만 등장
- [ ] 3분 이후: 엘리트 등장
- [ ] 동시 적 수 500마리 이하 유지 (maxEnemies 캡)
- [ ] 화면 밖에서만 스폰되는지 확인

### 세이브/로드
- [ ] 런 종료 후 골드 누적 저장
- [ ] 게임 재시작 후 골드 값 유지
- [ ] 영구 업그레이드 구매 후 재시작해도 유지
- [ ] 세이브 파일 삭제 후 기본값으로 정상 시작

## 3. 밸런스 조정 포인트

### 적 스케일링 (EnemyData)
```
hpScalePerMinute:  0.10 → 분당 HP +10%
  너무 쉬우면: 0.15~0.20으로 올림
  너무 어려우면: 0.05로 낮춤

speedScalePerMinute: 0.01
  후반 벽 느낌이면: 0.005
```

### 스폰 커브 (EnemyDirector.spawnRateCurve)
```
권장 기준점:
  0분  → 0.5/초
  3분  → 2.0/초
  10분 → 6.0/초
  18분 → 12.0/초
```

### XP 테이블 (PlayerStats.BuildXpTable)
```
현재: 5 * (level ^ 1.5)
레벨업 너무 빠르면 지수 1.7~2.0으로 올림
레벨업 너무 느리면 계수 5→3으로 낮춤
```

### 레벨업 선택지 풀
```
초반(1~5레벨): 무기 위주 나오게 가중치 조정 권장
중반(6~15레벨): 패시브 비중 증가
후반(16레벨~): 이미 갖춘 아이템 레벨업 위주
→ LevelUpUI.GenerateOptions()에 레벨 기반 가중치 추가 예정
```

### 진화 타이밍
```
목표: 8~12분 사이에 첫 진화 가능
  무기 MAX 레벨 8 → 평균 8레벨업 필요
  패시브 MAX 레벨 5 → 평균 5레벨업 필요
  합계 ~13 레벨업, 분당 ~1.3레벨업 시 10분
```
